What languages I have learned so far during my Web Development course:

CSS

Learning CSS has been a joy. Its ability to beautify websites with simple code is captivating. In this web development module, I was introduced to the fascinating world of CSS. Initially it has been very difficult due to the variety of functions and values, as I struggled to remember them all, but I eventually managed it! CSS isn't forgiving though, as a small mistake can wreck an entire website, a lesson I learned the hard way! Nonetheless, it has been a great experience.

For example, I've incorporated CSS keyframe animations into the background because I thought the animated aesthetic would give the site a modern and simplistic feel. I specifically chose red, blue, and black as these are my favorite colors overall. The choice of the little saturns (modified with CSS of course) on the navigation bar came from my passion for spacial bodies, and also because they look simplistic and modern with the black backgrounds!

This is how I achieved on creating such a background!

HTML

HTML will always be my favorite because it's so easy to handle, and it is a very forgiving language too! Since I first made my first project with pure HTML, being a simple red pill/blue pill selection page, I knew this language would definitely be my favourite. Again, I was introduced to this markup language on the exact same module, web development, and it was been my absolute pleasure to learn it.

Across the module, I have learnt how to apply good practice when creating semantic tags, divisors, headers, and paragraphs by using HTML and it has been a great experience.
